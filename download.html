Using the gem

Automatically vendors the latest version of batman.js for Rails 3.2 in your project. In the Gemfile for your project add gem "batman-rails".

After modifying your gemfile:

Requires Rails 3.2, and the rails asset pipeline provided by sprockets.

  • bundle install
    rails g batman:install
  • rake batman:app to set up the project
